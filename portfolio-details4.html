Project information

  • Recognize, Eliminate, and Contact Trace (REaCT)
  • Capstone Project
  • Android | CSS | Firebase | HTML | Javascript | Jquery | PHP | Kairos API
  • Project date : June 2021 - March 2022
  • Git: github.com/Evanchii/Recognize-Eliminate-and-Contact-Trace-Web

This is a School Project in ITE 309 Capstone and Research 1 & ITE 310 Capstone and Research 2

Worked with a team to design and develop a mobile app, web system, and database. The project aims to analyze the current performance of the existing contact tracing system and provide an IT Based solution to speed up the current manual processes of filling out contact tracing sheets and to make it quicker to stop the spread of the coronavirus disease (COVID-19).

  • Ensures user-friendly, responsiveness, and consistent performance for the users
  • Display the daily news about COVID-19
  • Won Best Capstone Project of the Year for System Development Track Year 2022
